# Asset Attribution

## HDR Environment Maps

Download HDR environment maps from [Poly Haven](https://polyhaven.com/hdris) (CC0 license)
and place `.hdr` files into `assets/env/`.

Recommended HDRIs for testing:
- `newport_loft` — indoor studio lighting
- `abandoned_parking` — overcast outdoor
- `kloppenheim_06` — bright outdoor with sun

All Poly Haven assets are released under CC0 (public domain).
